Detailed Description

Algorithm class performs Breadth First Search (BFS) operations on maze discovered in simulator.

Algorithm class is designed to analyze the maze file discovered from the simulator and find a path from the start location to goal location. The BFS algorithm will evaluate the precense of a wall to the neighboring cells to determine a clear path. The accessible cell locations around the current position are added to a queue of points to be evaluated. This process continues for each search "branch" produced by the search algorithm until an evaluated point reaches the goal area.
As the search algorithm produces a path to the goal area, the robot is commanded to move through the visited point list towards the goal. Robot movement commands are acheived through the API commands in a separate function.
solve(): Checks if an evaluated point has reached the goal location and traces final path produced from BFS algorithm.
generatePath(): Runs BFS algorithm to search and evaluate points while producing a suitable path to the goal.
followPath(): Performs checks to verify robot can move to next point among accessbile points.
moveRobot(): Performs commands to API functions for robot to move through maze.
printPath(): Displays path discovered from BFS algorithm.
colorPath(): Sets different color to finalized path in simulation display.
